# Справочник констант (`src/sim/config.ts`)

## Мир и среда

| Константа | Значение | Влияние |
|-----------|----------|---------|
| `WORLD.W` | 300 | Ширина сетки |
| `WORLD.H` | 200 | Высота сетки |
| `WORLD.SOIL_Y` | 124 | Ряд поверхности почвы |
| `SHADE_PER_LAYER` | 0.28 | Потеря света на слой затенения |
| `MAX_SHADE_LAYERS` | 4 | Больше эффективных слоёв → света 0 |
| `SHADE_SCATTER_INTERVAL` | 20 | Интервал рассеивания тени в колонке |
| `SHADED_SPROUT_LAYERS` | 5 | Порог затенённой мерistemы |
| `STEM_PHOTO_GAIN_FACTOR` | 0.2 | Доля фотосинтеза ствола от мерistemы |
| `DOUBLE_GROW_COST_MULT` | 3 | Множитель стоимости двойного GROW |
| `BASE_LIGHT` | 1.0 | Базовый поток света (растёт с высотой, см. `altitudeLight`) |
| `WATER_PER_DEPTH` | 0.25 | Вода с глубиной в почве |
| `AIR_WATER` | 0.05 | Влага в воздухе |
| `MIN_WATER_FOR_GROW` | 0.1 | Минимум воды для роста |
| `SURFACE_SOIL_WATER_FRAC` | 0.75 | Доля воды для ростка прямо над почвой |
| `DEATH_ENERGY_RETURN` | 0.30 | Доля энергии клетки → перегной |

## Клетки и энергия

| Константа | Значение | Влияние |
|-----------|----------|---------|
| `MAINTAIN.ROOT` | 0.035 | Расход на корень |
| `MAINTAIN.STEM` | 0.04 | Расход на ствол |
| `MAINTAIN.SPROUT` | 0.06 | Расход на росток-меристему |
| `MAINTAIN.SEED` | 0.03 | Расход на семя |
| `CAP.*` | 10–12 | Ёмкость буфера клетки |
| `TRANSPORT_RATE` | 0.35 | Скорость перетекания энергии |
| `GROW_COST` | 2.0 | Стоимость роста новой клетки |
| `GROW_MIN_ENERGY_RESERVE` | 5 | Подушка энергии до активного роста |
| `GROW_ENERGY_RESERVE_TICKS` | 2.5 | Резерв содержания (× upkeep) перед ростом |
| `ENERGY_PER_EXTRA_GROW_ACTION` | 14 | +1 ростовое действие за N ед. энергии сверх резерва |
| `SENESCENCE_START` | 0.72 | Доля maxAge, после которой растёт upkeep |
| `SENESCENCE_UPKEEP_GAIN` | 2.5 | Сила роста upkeep к концу жизни |
| `PHOTO_GAIN_FACTOR` | 1.5 | Множитель фотосинтеза |
| `MINERAL_SUPPLY_NORM` | 1.6 | Нормировка минерального снабжения (закон Либиха) |
| `MINERAL_ENERGY_FACTOR` | 0.4 | Конверсия минералов в энергию |
| `SPROUT_SINK_POTENTIAL` | 0.15 | Притяжение энергии к растущим росткам |

Фотосинтез ограничен по закону Либиха: минимум из света и минерального
снабжения. Без корней (минералов) и без света побег не накапливает энергию.

## Минералы

| Константа | Значение | Влияние |
|-----------|----------|---------|
| `MINERAL_BASE` | 8 | Стартовый запас у поверхности |
| `MINERAL_DEPTH_GAIN` | 0.55 | Прирост с глубиной |
| `MINERAL_CAP` | 40 | Потолок в ячейке |
| `MINERAL_SINK_RATE` | 0.02 | Просачивание вниз / сток |
| `MINERAL_DIFFUSE_RATE` | 0.05 | Растекание вбок |
| `MINERAL_REGEN_RATE` | 0.01 | Восстановление к базовому уровню (выветривание/осадки) |
| `UPTAKE_MAX` | 1.2 | Макс. добыча корнем за тик |

Регенерация минералов критична для устойчивости: без неё бум-поколение выедает
приповерхностный слой, а следующее голодает и популяция вымирает.

## Размножение и симуляция

| Константа | Значение | Влияние |
|-----------|----------|---------|
| `INITIAL_PLANTS` | 30 | Стартовое число растений |
| `SEED_GERMINATION_TICKS` | 10 | Задержка прорастания |
| `MIN_SEED_ENERGY` | 5 | Мин. энергия семени для прорастания (ниже — сразу гибнет) |
| `SEED_SOIL_UPKEEP` | 0.008 | Расход энергии в почве после окна ожидания (за тик) |
| `SEED_MIN_PAYLOAD` | 1 | Мин. энергия в только что созданном семени |
| `SEED_FORMATION_OVERHEAD` | 3 | Доп. затраты на образование семени (не входят в payload) |
| `SPIKE_COST` | 1.5 | Энергия на выставление шипа |
| `SHOOT_COST` | 2.5 | Энергия на «выстрел» шипа на 2 клетки |
| `SPIKE_LEAF_KILL_RADIUS` | 2 | Радиус гибели чужих листьев вокруг шипа |
| `MAX_GERMINATIONS_PER_TICK` | 4 | Лимит прорастаний за тик |
| `MIN_PLANT_SPACING` | 5 | Мин. дистанция по X между стартовыми/проросшими растениями |
| `SEED_FALL_DURATION_TICKS` | 4 | Длительность падения семени (тики) |
| `MIN_SEED_HEIGHT` | 2 | Семя — только с побега не ниже этой высоты (закон движка) |
| `SEED_SCATTER` | 1 | Макс. смещение семени по X от места создания |

Ключ к устойчивости популяции — баланс между **ранними лёгкими семенами**
(`SEED` с малым аргументом доли энергии) и **крупными** (аргумент близок к 1).
Семя в почве и на побеге тратит `MAINTAIN.SEED` каждый тик; при `energy <= 0`
исчезает. Прорастание требует `MIN_SEED_ENERGY`.

Все значения можно менять в `config.ts` для балансировки.
